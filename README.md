# starship-prompt

1. install Starship  https://starship.rs/
   Read Instructions for your Shell ( bash | zsh | fish )

2. Clone Repository
```
git clone https://github.com/fkessler1/starship-prompt.git
```



3. Create Symlink 
``` 
cd starship-prompt && \
ln -sf $PWD/config/starship.toml ~/.config/starship.toml
```
